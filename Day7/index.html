CSS Selectors

    Basic Selectors :

  1. Universal Selector : (*)
  2. Type Selector : directly tag will selected (e.g, p a, h1, etc)
  3. Class Selector : (.classname)
  4. ID Selector : (#id)
  5. Group Selector : multiple tag, class, or id selected using "," (A, B)
  6. Descendant Selector : selets all elements "B" inside "A" (A B)
  7. Child Selector : selects all child element "B" indide "A" (A>B)
  8. Adjacent Sibling Selector : selects the next sibling "B "A" (A+B)
  9. General Sibling Selector : selects all siblings (A ~ B)
CSS pseudo-classes

pseudo-classes target elements based on their state, position, or user interaction. They are weitten with a colon (:) before the keyword

  1. :hover :- Applies styles when the user hovers over an element.
  2. :focus :- Styles an element when it gains focus (e.g., clicking or tabbing into an input field)
  3. :nth-child(n) :- Targets the nth of a parent.
  4. :first-child and :last-child :- Targets the first and last children of a parent.
  5. :not(selector) :- Exclude elements that match the selector.
