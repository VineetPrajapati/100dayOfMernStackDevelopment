Introduction to CSS

CSS (Cascading Style Sheet) is used to design the web page. With css we can control color, font, spcing and more and it allow us to make the web page visually appealing website.

Basic of Styling

There are 3 type of styling in Css.

  1. Inline CSS : The styling added with the tag
  2. Internal CSS : The styling added within the HTML document in head tag by using style tag
  3. External CSS : In external css seperate .css file added using link tag within head/
CSS Selector
  1. Element Selector : tag i.e h1, p, a, span etc. is selcted for the designing
  2. Class Selector : using .className
  3. ID Selector : using #id
  4. Universal Selector : It is used to select he entire page by uing "*"
