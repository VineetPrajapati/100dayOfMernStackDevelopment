Responsive desig ensures that web pages looks good and are usable on devices of all screen sizes, including desktops, tablests, and mobile phones.

    Techniques for responsive desig include :

  • Flexible layouts using percentages or relative units (e.g., em, rem, %)
  • Responsive images that scales with their container
  • CSS media queries to apply styles conditionally based on the viewport size.

Media Queries

Media queries are a CSS features that allows us to apply styles conditionally bassed on the properties of the uses's devices, such as size, resolution, or orientation.

Common Media Query Breakpoints

Device Type Breakpoints
Extra Small (Mobile) 0px - 575px
Small (Tablet) 576px - 767px
Medium (Small Laptop) 768px - 991px
Large (Desktop) 992px - 1199px
Extra Large (Wide Desktop) 1200px and above

Types of Media Queries

  1. Width-Based Queries : Apply styles based on the screen width.
  2. Orientation-Based Queries : Apply styles based orientation (portrait or landscape).
  3. Resolution-Based Queries : Target devices based on their screen resolution
