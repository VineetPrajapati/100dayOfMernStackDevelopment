CSS Box Model

Box model contains :

  • Content : The area where text, image, or other content is displayed
  • Padding : Space between the content and the border
  • Border : A line that wrap around the the padding and content
  • Margin : Space outside the border, creating distance between the element and element

Positioning

CSS positioning allows us to control the placement of element within a webpage. There are five main position values static, relative, absolute, fixed, and sticky

position static is the default vlaue of all element

position relative to its normal position, slightly changes are possible such as top, right, bottom, and left

In absolute the positions the element relative to the nearest positioned ancestor (i.e, any parent with position set to relative, absolute, or fixed)
