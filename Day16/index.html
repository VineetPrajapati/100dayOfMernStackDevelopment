Day 16: Event handling basics in JavaScript (click events, input events).

JavaScript "events" allow us to make our web pages interactive by responding to user actions like "click", "typing", "scrolling", and more. Today, we will focus onl click events, input events, and how to use them effectively

1. What Are Events?

Events represent user interactions or browser actions, such as:

  • Clicking a button
  • Typing in a text field
  • Hovering over an element
  • Loading a webpage

Event handling refers to the process of responding to these events

2. Adding Event Listeners

To handle events in JavaScript, we have to attach an event Listeners to an HTML element. When the event occurs, the listener executes a callback function.
